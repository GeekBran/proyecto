Tienes Derecho a Conocerte

¿Quiénes Somos?



Somos una plataforma que funciona como herramineta y alternativa para resolver el tan tedioso problema de lidiar con las esperas a la hora de realizar
trámites en las instituciones médicas aún incluso cuando solo necesitamos consultar un único dato.













¿Qué ofrecemos?



Que puedas visualizar tus datos por medio de este sitio a través de una cuenta personal para tener acceso a tus datos, que serán confidenciales
por supuesto, consultando tu expediente clínico para visulizar todo tu historial, o realizar una copia de seguridad e imprimirlo.
Además, puedes agendar citas, ya sean consulta general o para rayos x, así mismo puedes consultar el seguimiento de alguna enfermedad o tratamiento que
estés llevando, alguna rehabilitación o incluso tus vacunas.













¿Por qué lo hacemos?



Consideramos que cada persona tiene derecho a conocerse, ya que muchos ignoramos cosas importantes como: tipo de sangre, alergias, enfermedades hereditarias, etc.
o simplemente olvidamos información de atenciones anteriores. Es por esto que se realizó este sitio para apoyar en estas situaciones.
